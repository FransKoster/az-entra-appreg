# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-05

### Added
- Initial release of Azure Entra ID Application Registration Terraform module
- Support for creating multiple application registrations
- Support for creating associated service principals (enterprise applications)
- Support for web applications with OAuth2 implicit grant
- Support for single page applications (SPA)
- Support for public client applications
- Support for API applications with OAuth2 permission scopes
- Support for app roles
- Support for required resource access configuration
- Support for optional claims
- Comprehensive examples for common use cases
- Full documentation in README.md
